# Repositório Unificado de Projetos Java

Este repositório reúne diversos projetos Java organizados em subpastas, cada um com seu próprio domínio de aplicação. O objetivo é facilitar o versionamento, manutenção e evolução dos projetos em um único local.

## Estrutura dos Projetos

- **com.example.agencia**: Sistema para gestão de uma agência de locação de equipamentos, veículos e peças streetwear.
- **com.example.banco**: Simulação de operações bancárias, incluindo contas, transações e auditoria.
- **com.example.embarque**: Gerenciamento de embarque aéreo, fidelidade e controle de passagens.
- **com.example.estudio**: Controle de estúdio fotográfico e audiovisual, com ensaios, sessões e relatórios de serviço.
- **com.example.gestaocontratos**: Gestão de contratos, reservas e salas de reunião.

Cada projeto possui sua própria estrutura de pastas, separando domínio, serviços, exceções e testes.
